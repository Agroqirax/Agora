## Updates

### Controleer op updates

Tik op **Controleren op updates** om GitHub te vragen naar de nieuwste release. Agora vergelijkt uw geïnstalleerde versie met de nieuwste release en toont het resultaat:

- **Update beschikbaar** — er is een nieuwere versie gepubliceerd op GitHub
- **Up-to-date**: u gebruikt de nieuwste release

### Automatische updatecontrole

Indien ingeschakeld, controleert Agora periodiek op nieuwe releases op de achtergrond. Schakel dit uit om automatische controles uit te schakelen.

---

## Documentatie

### Documentatielinks tonen

Bestuurt de **zwevende helpknop** (FAB) die verschijnt op de subpagina's van de instellingen. Indien ingeschakeld, linkt een :material-help-circle:-knop naar de relevante gebruikershandleidingpagina.

Schakel dit uit als u de voorkeur geeft aan een schonere interface zonder de documentatie FAB.

## Crashlogboek

Als Agora een fout tegenkomt, kunt u het crashlogboek bekijken via **Instellingen → Info → Crashlog**. Het logboek wordt weergegeven in een monospace-dialoogvenster met een knop **Kopiëren** om diagnostische details te delen bij het melden van bugs.
