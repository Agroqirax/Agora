# Taal

Kies de weergavetaal voor de interface van Agora.

## Beschikbare talen

| Taal | Code | Beschrijving |
|----------|------|-------------|
| **Systeemstandaard** | `systeem` | Volg de systeemtaalinstelling van uw apparaat |
| **Engels** | `nl` | Alle UI-tekst in het Engels |
| **简体中文** | `zh` | Alle UI-tekst in Vereenvoudigd Chinees |
| **繁體中文** | `zh-Hant` | Alle UI-tekst in traditioneel Chinees |

## Hoe te veranderen

1. Ga naar **Instellingen → Taal**
2. Selecteer uw voorkeurstaal
3. Er verschijnt een snackbalk: tik op **Herstart** om de wijziging onmiddellijk toe te passen

!!! let op "Opnieuw opstarten vereist"
    Taalwijzigingen vereisen een volledige herstart van de app. Agora vraagt ​​u om opnieuw op te starten nadat u een nieuwe taal hebt geselecteerd. Tik op de herstartactie in de snackbar om de app opnieuw te laden met de nieuwe taal.
