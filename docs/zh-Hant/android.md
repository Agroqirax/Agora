# Android 工具

Agora 可在裝置需要時安全地存取選定的 Android 系統功能。這些工具允許裝置在遵守 Android 權限系統的前提下取得您的當前位置、讀取聯絡人或與您的日曆互動。

## 可用工具

| 工具 | 用途 |

| ------------ | ----------------------------------------------------- |

| **位置** | 取得設備的近似或精確位置 |

| **聯絡人** | 搜尋並讀取裝置上儲存的聯絡人 |

| **日曆** | 讀取即將到來的事件並建立新的日曆條目 |

設備會自動發現已啟用的工具，並判斷它們在對話過程中何時有用。

## 隱私與權限

Android 工具需要標準的 Android 運行時權限。

當裝置首次嘗試使用這些工具時，Agora 將要求對應的 Android 權限。權限僅在首次需要使用工具時才會要求。

!!! 注意

您可以隨時在裝置的 Android 設定中撤銷權限。

## 設定

1. 前往**設定 → Android**

2. 啟用您希望模型使用的工具：

- **位置資訊**

- **聯絡人**

- **日曆**

3. 出現提示時，授予所需的 Android 權限。

啟用後，模型可以在對話過程中根據需要自動存取這些工具。

## 位置資訊

位置資訊工具可讓模型確定您設備的當前位置。

典型用途包括：

- 尋找附近地點

- 提供當地天氣資訊

- 位置感知推薦

- 估算行程時間

- 回答有關您目前所在區域的問題

根據您的裝置設定和授予的權限，位置資訊可能為近似值或精確值。

## 聯絡人

聯絡人工具可讓模型搜尋儲存在您裝置上的聯絡人。

典型用途包括：

- 尋找電話號碼

- 尋找電子郵件地址

- 識別已儲存的聯絡人

- 選擇聯絡人以執行訊息或通訊任務

此模型僅存取完成您的請求所需的聯絡人資訊。

## 日曆

日曆工具允許該模型讀取您的日曆並建立事件。

典型用途包括：

- 查看您的日程安排

- 列出即將發生的事件

- 尋找可用時間

- 建立約會

- 查看會議詳情

建立或修改事件需要日曆寫入權限。

## 安全性

Android 工具使用 Android 內建的權限系統。

- 首次使用前會顯示運行時權限提示

- 權限可隨時撤銷

- 已停用的工具無法存取

- 所有存取均透過 Android 的權限框架在本地進行

未經您的許可，Agora 無法存取受保護的資料。

## 故障排除

### 權限被拒絕

如果設備報告無法存取某個工具：

- 確認工具已在**設定 → Android** 中啟用

- 確認已授予所需的 Android 權限

- 如有必要，請在 Android 設定中撤銷並重新授予該權限

### 位置不可用

- 確保您的裝置已啟用位置服務

- 移動到 GPS 或網路覆蓋更好的區域

- 如果需要更高精度的位置信息，請授予精確位置權限

### 日曆或聯絡人為空

請確認您的裝置包含日曆事件或聯絡人，並已授予相應的 Android 權限。
