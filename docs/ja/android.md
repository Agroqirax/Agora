# Androidツール

Agoraは、モデルが必要とする場合に、選択されたAndroidシステム機能に安全にアクセスできます。これらのツールを使用すると、モデルはAndroidの権限システムを尊重しながら、現在地を取得したり、連絡先を読み取ったり、カレンダーを操作したりできます。

## 利用可能なツール

| ツール | 用途 |

| ------------ | ----------------------------------------------------- |

| **位置情報** | デバイスのおおよその位置または正確な位置を取得します |

| **連絡先** | デバイスに保存されている連絡先を検索して読み取ります |

| **カレンダー** | 今後の予定を読み取り、新しいカレンダーエントリを作成します |

モデルは、有効になっているツールを自動的に検出し、会話中にそれらが役立つタイミングを判断します。

## プライバシーと権限

Androidツールには、標準のAndroidランタイム権限が必要です。

モデルがこれらのツールを初めて使用しようとすると、Agoraは適切なAndroid権限を要求します。権限は、ツールが初めて必要になったときにのみ要求されます。

!!!注

デバイスのAndroid設定から、いつでも権限を取り消すことができます。

## 設定

1. **設定 → Android** に移動します。
2. モデルに使用させたいツールを有効にします。

- **位置情報**

- **連絡先**

- **カレンダー**

3. プロンプトが表示されたら、要求されたAndroid権限を許可します。

有効にすると、モデルは会話中に役立つ場合にこれらのツールに自動的にアクセスできるようになります。

## 位置情報

位置情報ツールを使用すると、モデルはデバイスの現在位置を特定できます。

一般的な使用例：

- 周辺の場所を検索
- 現地の天気情報を提供
- 位置情報に基づいたおすすめ情報を提供
- 移動時間の推定
- 現在地に関する質問に回答

デバイスの設定と許可された権限によっては、位置情報は概算値または高精度になります。

## 連絡先

連絡先ツールを使用すると、モデルはデバイスに保存されている連絡先を検索できます。

一般的な用途は以下のとおりです。

- 電話番号の検索
- メールアドレスの検索
- 保存済みの連絡先の特定
- メッセージ送信やコミュニケーションタスクに使用する連絡先の選択

このモデルは、リクエストの処理に必要な連絡先情報のみにアクセスします。

## カレンダー

カレンダーツールを使用すると、モデルはカレンダーを読み取り、イベントを作成できます。

一般的な用途は以下のとおりです。

- スケジュールの確認
- 今後のイベント一覧の表示
- 空き時間の検索
- 予定の作成
- 会議の詳細の確認

イベントの作成または変更には、カレンダーへの書き込み権限が必要です。

## セキュリティ

Androidツールは、Androidの組み込み権限システムを使用します。

- 初回使用前に実行時権限プロンプトが表示されます
- 権限はいつでも取り消すことができます
- 無効化されたツールにはアクセスできません
- すべてのアクセスは、Androidの権限フレームワークを通じてローカルで行われます

Agoraは、お客様の許可なしに保護されたデータにアクセスすることはできません。

## トラブルシューティング

### 権限が拒否されました

端末がツールへのアクセスを拒否した場合：

- **設定 → Android** でツールが有効になっていることを確認してください。
- 必要な Android 権限が付与されていることを確認してください。
- 必要に応じて、Android 設定で権限を取り消し、再度付与してください。

### 位置情報が利用できません

- 端末で位置情報サービスが有効になっていることを確認してください。
- GPS またはネットワークの電波状況が良い場所に移動してください。
- より高精度の位置情報が必要な場合は、正確な位置情報を許可してください。

### カレンダーまたは連絡先が空です

端末にカレンダーの予定または連絡先が登録されており、対応する Android 権限が付与されていることを確認してください。
