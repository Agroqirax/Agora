# 안드로이드 도구

Agora는 필요할 때 선택된 안드로이드 시스템 기능에 안전하게 접근할 수 있습니다. 이러한 도구를 통해 모델은 안드로이드 권한 시스템을 준수하면서 사용자의 현재 위치를 가져오거나, 연락처를 읽거나, 캘린더와 상호 작용할 수 있습니다.

## 사용 가능한 도구

| 도구 | 용도 |

| ------------ | ----------------------------------------------------- |

| **위치** | 기기의 대략적인 또는 정확한 위치 가져오기 |

| **연락처** | 기기에 저장된 연락처 검색 및 읽기 |

| **캘린더** | 예정된 일정 읽기 및 새 캘린더 항목 추가 |

모델은 활성화된 도구를 자동으로 검색하고 대화 중에 유용하다고 판단되는 시점을 결정합니다.

## 개인정보 보호 및 권한

안드로이드 도구는 표준 안드로이드 런타임 권한을 필요로 합니다.

모델이 이러한 도구 중 하나를 처음 사용하려고 할 때 Agora는 해당 안드로이드 권한을 요청합니다. 권한은 도구가 처음 필요할 때만 요청됩니다.

!!! 참고
기기의 안드로이드 설정에서 언제든지 권한을 취소할 수 있습니다.

## 설정

1. **설정 → Android**로 이동합니다.
2. 모델이 사용할 도구를 활성화합니다.

- **위치**

- **연락처**

- **캘린더**

3. 메시지가 표시되면 요청된 Android 권한을 허용합니다.

활성화되면 모델은 대화 중 필요할 때마다 이러한 도구에 자동으로 접근할 수 있습니다.

## 위치

위치 도구를 사용하면 모델이 기기의 현재 위치를 파악할 수 있습니다.

일반적인 사용 예는 다음과 같습니다.

- 주변 장소 찾기
- 지역 날씨 정보 제공
- 위치 기반 추천
- 예상 이동 시간
- 현재 지역에 대한 질문에 답변

기기 설정 및 허용된 권한에 따라 위치 정보가 대략적이거나 정확할 수 있습니다.

## 연락처

연락처 도구를 사용하면 모델이 기기에 저장된 연락처를 검색할 수 있습니다.

일반적인 사용 예는 다음과 같습니다.

- 전화번호 조회
- 이메일 주소 찾기
- 저장된 연락처 확인
- 메시지 또는 커뮤니케이션 작업에 사용할 연락처 선택

모델은 사용자의 요청을 처리하는 데 필요한 연락처 정보에만 접근합니다.

## 캘린더

캘린더 도구를 사용하면 모델이 사용자의 캘린더를 읽고 이벤트를 생성할 수 있습니다.

일반적인 사용 예는 다음과 같습니다.

- 일정 확인
- 예정된 이벤트 목록 보기
- 사용 가능한 시간 찾기
- 약속 생성
- 회의 세부 정보 검토

이벤트를 생성하거나 수정하려면 캘린더 쓰기 권한이 필요합니다.

## 보안

Android 도구는 Android의 내장 권한 시스템을 사용합니다.

- 처음 사용하기 전에 런타임 권한 요청 메시지가 표시됩니다.
- 권한은 언제든지 취소할 수 있습니다.
- 권한이 비활성화된 도구는 접근할 수 없습니다.
- 모든 접근은 Android 권한 프레임워크를 통해 로컬에서 이루어집니다.

Agora는 사용자의 허가 없이는 보호된 데이터에 접근할 수 없습니다.

## 문제 해결

### 권한 거부

기기에서 도구에 접근할 수 없다는 메시지가 표시되면 다음을 확인하세요.

- **설정 → Android**에서 해당 도구가 활성화되어 있는지 확인합니다.
- 필요한 Android 권한이 부여되었는지 확인합니다.
- 필요한 경우 Android 설정에서 권한을 취소한 후 다시 부여합니다.

### 위치 정보 없음

- 기기에서 위치 서비스가 활성화되어 있는지 확인합니다.
- GPS 또는 네트워크 신호가 더 잘 잡히는 지역으로 이동합니다.
- 더 정확한 위치 정보가 필요한 경우 정확한 위치 권한을 부여합니다.

### 캘린더 또는 연락처가 비어 있음

기기에 캘린더 일정이나 연락처가 있는지, 그리고 해당 Android 권한이 부여되었는지 확인합니다.
