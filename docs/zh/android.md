# Android 工具

Agora 可在设备需要时安全地访问选定的 Android 系统功能。这些工具允许设备在遵守 Android 权限系统的前提下获取您的当前位置、读取联系人或与您的日历进行交互。

## 可用工具

| 工具 | 用途 |

| ------------ | ----------------------------------------------------- |

| **位置** | 获取设备的近似或精确位置 |

| **联系人** | 搜索并读取设备上存储的联系人 |

| **日历** | 读取即将到来的事件并创建新的日历条目 |

设备会自动发现已启用的工具，并判断它们在对话过程中何时有用。

## 隐私与权限

Android 工具需要标准的 Android 运行时权限。

设备首次尝试使用这些工具时，Agora 将请求相应的 Android 权限。权限仅在首次需要使用工具时才会请求。

!!! 注意

您可以随时在设备的 Android 设置中撤销权限。

## 设置

1. 前往**设置 → Android**

2. 启用您希望模型使用的工具：

- **位置信息**

- **联系人**

- **日历**

3. 出现提示时，授予所需的 Android 权限。

启用后，模型可以在对话过程中根据需要自动访问这些工具。

## 位置信息

位置信息工具允许模型确定您设备的当前位置。

典型用途包括：

- 查找附近地点

- 提供当地天气信息

- 位置感知推荐

- 估算行程时间

- 回答有关您当前所在区域的问题

根据您的设备设置和授予的权限，位置信息可能为近似值或精确值。

## 联系人

联系人工具允许模型搜索存储在您设备上的联系人。

典型用途包括：

- 查找电话号码

- 查找电子邮件地址

- 识别已保存的联系人

- 选择联系人以执行消息或通信任务

该模型仅访问完成您的请求所需的联系人信息。

## 日历

日历工具允许该模型读取您的日历并创建事件。

典型用途包括：

- 查看您的日程安排

- 列出即将发生的事件

- 查找可用时间

- 创建约会

- 查看会议详情

创建或修改事件需要日历写入权限。

## 安全性

Android 工具使用 Android 内置的权限系统。

- 首次使用前会显示运行时权限提示

- 权限可随时撤销

- 已禁用的工具无法访问

- 所有访问均通过 Android 的权限框架在本地进行

未经您的许可，Agora 无法访问受保护的数据。

## 故障排除

### 权限被拒绝

如果设备报告无法访问某个工具：

- 确认该工具已在**设置 → Android** 中启用

- 确认已授予所需的 Android 权限

- 如有必要，请在 Android 设置中撤销并重新授予该权限

### 位置不可用

- 确保您的设备已启用位置服务

- 移动到 GPS 或网络覆盖更好的区域

- 如果需要更高精度的位置信息，请授予精确位置权限

### 日历或联系人为空

请确认您的设备包含日历事件或联系人，并且已授予相应的 Android 权限。
